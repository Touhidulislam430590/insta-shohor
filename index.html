How does JavaScript work?

JavaScript is a scripting language what specially designed for client-site web page. JavaScript is a runtime load data platform. Every browser has build-in javascript interpreter like google chorme has 'V8', Edge has 'Chakra', Safari has 'JavaScript Core' and Firefox has 'Spidermonkey'. JavaScript load html elements by using javascript event loop. Interpreter and event loop's combination makes JavaScript more beauty and powerfulBasically event loop works 3 steps. that's are Stack, Queue and Heap. Stack loads all frames, Queue loads all messages what to display which event load after which element and finally Heap loads all the objects as like tree.



Difference between Local storage and Session storage

Local Storage :

Local storage is a browser storage which used basically store user name and user password. Sometimes it used for stored cart data. The benifit of local storage is there have no expiration date. local storage saved data as key-value.

Session Storage :

Session storage is a virtual storage that build-in on browser. Sometimes we need to store some data in runtime session. That means stored data will be deleted when the browser tab close. But if you reload the tab then session storage still stored data.
